## 🌟  Git Branch

Creating a new branch:

#### 🌓 git branch

To push the new branch into the remote repository, you need to use the folowing commmand:

#### 🚀 git push -u

Viewing branchs: 

#### 🌟 git branch or git branch --list

Deleting a branch:

#### 🎇 git branch -d
